# 供应链审计与内控实践

> 基于在大型制造企业（不锈钢/新能源产业链）的审计工作经验，总结供应链内控体系。

## 一、审计范围

制造企业供应链覆盖从矿山到产成品的全链条：

```
矿产采购 → 物流运输 → 仓储管理 → 生产领用 → 产成品销售 → 资金回笼
   ↑            ↑          ↑          ↑           ↑           ↑
 采购审计    物流审计   库存审计   成本审计    销售审计    资金审计
```

## 二、采购审计

### 2.1 核心风险

| 风险点 | 表现 | 审计方法 |
|--------|------|---------|
| 围标串标 | 供应商轮流中标 | 投标 IP / 投标文件比对 |
| 虚假采购 | 采购未实际交付 | 三单匹配（合同+入库+发票） |
| 利益输送 | 采购价明显高于市场 | 价格对标（同期/同行业） |
| 供应商垄断 | 单一来源占比过高 | 供应商集中度分析 |
| 资质造假 | 供应商虚构资质 | 穿行测试 + 实地考察 |

### 2.2 价格审计方法

**市场对标法：**
```
价格偏离度 = (采购价 - 市场基准价) / 市场基准价
红色预警：偏离度 > 5%
黄色预警：偏离度 > 3%
```

**历史趋势法：**
- 同品种连续12个月价格走势
- 环比/同比异常波动检测
- 季节性规律验证

### 2.3 供应商管理审计

- 供应商准入流程是否完整（资质审查、实地考察、样品测试）
- 合格供应商名录是否定期更新
- 供应商绩效评价（交付及时率、质量合格率、价格竞争力）
- 黑名单/淘汰机制是否有效执行
- 关联方供应商是否披露完整

## 三、库存审计

### 3.1 盘点流程

| 步骤 | 内容 | 要点 |
|------|------|------|
| 盘点前 | 制定计划、通知仓库冻结 | 突击抽查 vs 定期盘点 |
| 盘点中 | 盲盘（审计员独立记录） | 不与仓库人员核对 |
| 差异分析 | 账实比对 | 差异 > 阈值 → 追溯 |
| 盘盈盘亏 | 原因分析和责任人 | 系统性 vs 偶发性 |

### 3.2 常见问题

- **虚增库存**：入库已记录但实物未到
- **以次充好**：质检等级与实际不符
- **呆滞库存**：超过 N 个月未周转未计提减值
- **账外物资**：寄存/代管物资未入账

### 3.3 关键指标

```
库存周转率 = 销售成本 / 平均库存
库龄分析：< 30天 / 30-90天 / 90-180天 / > 180天
呆滞率 = 呆滞库存 / 总库存
盘亏率 = 盘亏金额 / 库存总额
```

## 四、物流审计

### 4.1 运费审计

| 方法 | 操作 | 工具 |
|------|------|------|
| 运价对标 | 同类路线/距离的市场运价对比 | 物流平台数据 |
| 吨公里成本 | 运费/(吨数×距离) 同业对比 | 行业报告 |
| 空驶率检查 | GPS 轨迹与实际运单对比 | GPS 数据 + 运单 |
| 虚假运输 | 运单与过磅单/出入库单比对 | 三单匹配 |

### 4.2 大宗商品物流特殊性

- **途耗率管控**：镍矿/煤炭等散货运输途耗标准
- **检斤检质**：装港 vs 卸港的第三方检测对比
- **滞期费**：港口滞期的责任界定和成本控制
- **保税物流**：加工贸易手册核销合规

## 五、销售审计

### 5.1 信用风险控制

```
客户信用管理流程：
  资质审核 → 额度评估 → 账期审批 → 动态监控 → 逾期催收

监控指标：
  - 应收账款周转天数
  - 逾期账款占比
  - 坏账率
  - 前 5 大客户集中度
```

### 5.2 价格审计

- 销售价格与市场价格的偏离度
- 关联方交易定价是否公允（转移定价风险）
- 折扣/返利审批权限是否合规
- 长协价与现货价的价差合理性

## 六、成本审计

### 6.1 生产成本

| 要素 | 审计重点 | 方法 |
|------|---------|------|
| 原料成本 | 单耗是否合理 | 理论单耗 vs 实际单耗 |
| 能源成本 | 电耗/气耗异常 | 行业对标 + 历史趋势 |
| 人工成本 | 计件工资合理性 | 工时定额与产出匹配 |
| 制造费用 | 分摊是否合理 | 成本动因分析 |

### 6.2 成本结转

- 在产品与产成品的成本分配
- 副产品/废料的价值核算
- 成本计算方法的变更是否合理
- 存货跌价准备的计提是否充分

## 七、内控体系

### 7.1 不相容职务分离

| 环节 | 必须分离的岗位 |
|------|--------------|
| 采购 | 请购 → 采购 → 验收 → 付款 |
| 销售 | 定价 → 发货 → 开票 → 收款 |
| 仓库 | 保管 → 记账 → 盘点 |
| 财务 | 出纳 → 记账 → 审核 |

### 7.2 审批权限矩阵

```
金额区间          采购审批      销售折让      费用报销
< 1万            部门经理      部门经理      部门经理
1-10万           分管副总      分管副总      财务总监
10-100万         总经理        总经理        总经理
> 100万          董事会        董事会        董事会
```

### 7.3 关键控制点（KCP）

- 采购合同审批 → 关注供应商选择依据
- 入库验收 → 关注质检报告和数量核实
- 付款审批 → 关注三单匹配
- 存货盘点 → 关注账实差异处理
- 销售定价 → 关注价格偏离度审批

## 八、数字化审计

### 8.1 数据分析工具

- **SQL / Python**：从 ERP（SAP/Oracle）抽取数据分析
- **Power BI / Tableau**：可视化 Dashboard
- **ACL / IDEA**：专业审计分析软件
- **RPA**：自动化重复性审计程序（如银行对账）

### 8.2 典型分析场景

1. **重复付款检测**：同一供应商、同一金额、相近时间的付款匹配
2. **异常付款检测**：周末/节假日付款、整数大额付款
3. **幽灵员工检测**：工资表与考勤/门禁数据比对
4. **拆分采购检测**：同一天、同一供应商的多笔小额订单

## 九、经验总结

1. **审计不是找茬，是找风险**——聚焦高风险领域，给业务提改进建议，不是单纯挑错。
2. **数据比人可靠**——能用系统跑出来的问题，不要靠人工抽查。
3. **了解业务才能审好**——不懂镍矿怎么定价、不锈钢怎么炼，就审不了采购和成本。
4. **突击盘点是杀手锏**——仓库永远不知道你什么时候来。
5. **审计报告要落地**——问题+根因+建议+整改期限+责任人，形成闭环。
