# 大宗商品套期保值风控实践

> 基于在大型制造业/大宗商品贸易企业的工作经验，总结套期保值业务的风险管理体系。

## 一、业务背景

大型制造企业（不锈钢/镍/新能源材料）需要对冲原材料价格波动风险：

- **买入套保**：锁定未来采购成本（如镍、铬铁、焦炭）
- **卖出套保**：锁定产成品销售价格（如不锈钢、硫酸镍）
- **库存套保**：对冲在手库存跌价风险

**核心原则：** 套保是为了对冲现货经营风险，不是为了投机获利。

## 二、风险识别

### 2.1 市场风险

| 风险类型 | 描述 | 实例 |
|---------|------|------|
| 价格风险 | 期货价格剧烈波动 | 2022年伦镍事件，镍价两日暴涨250% |
| 基差风险 | 现货与期货价差异常 | 交割月基差不收敛 |
| 流动性风险 | 极端行情无法平仓 | 涨跌停板限制 |
| 汇率风险 | 外盘套保的汇兑损失 | LME美元计价 vs 人民币成本 |

### 2.2 操作风险

- **保证金不足**：极端行情追保压力巨大（伦镍事件核心教训）
- **超量套保**：期货头寸超过现货敞口，实质构成投机
- **交割风险**：货源不足或品质不达标无法交割
- **交易对手风险**：经纪商/Margin Call 处理不当

### 2.3 合规风险

- 套保会计（IAS 39 / IFRS 9）合规要求
- 国资委/证监会关于国有企业套保的规定
- 跨境资金流动的外汇管理

## 三、风控体系

### 3.1 组织架构

```
董事会
  └─ 风控委员会
      ├─ 风控部（独立于业务部）
      │   ├─ 市场风险组
      │   ├─ 信用风险组
      │   └─ 操作风险组
      ├─ 套保业务部
      └─ 财务部（保证金管理）
```

### 3.2 套保额度管理

| 维度 | 控制标准 | 审批层级 |
|------|---------|---------|
| 总套保规模 | ≤ 现货敞口的 100% | 风控委员会 |
| 单品种上限 | ≤ 总敞口的 N% | 风控总监 |
| 单月开仓量 | 不超过月产量的 N% | 业务负责人 |
| 保证金占用 | ≤ 可用资金的 N% | 财务总监 |
| 止损限额 | 单日 ≤ X / 累计 ≤ Y | 交易员 |

### 3.3 实时监控指标

**每日监控看板：**

```
□ 期货持仓 vs 现货敞口匹配度
□ 保证金占用率（当前/预警/极限）
□ 浮动盈亏（逐日盯市）
□ 基差走势（当前 vs 历史区间）
□ 期权 Greek 值（如使用期权套保）
□ 经纪商集中度（前3家占比）
□ 交割月持仓（是否有逼仓风险）
```

### 3.4 压力测试

**情景设定：**

| 情景 | 假设 | 频率 |
|------|------|------|
| 轻度 | 价格波动 ±10% | 每周 |
| 中度 | 价格波动 ±25% + 基差扩大 | 每月 |
| 重度 | 价格波动 ±50% + 流动性枯竭 | 每季度 |
| 极端 | 类似 2022 伦镍事件 | 每半年 |

**测试输出：**
- 追加保证金需求
- 最大亏损额
- 流动性缺口
- 是否需要启动应急预案

## 四、伦镍事件复盘与教训

### 4.1 事件回顾

2022年3月，LME镍期货价格两日内从约3万美元/吨暴涨至超10万美元/吨，LME被迫暂停交易并取消数十亿美元交易。青山控股作为大空头，面临巨大追保压力。

### 4.2 核心教训

1. **仓位集中度过高**：单一品种、单一方向的头寸过大，不具备极端行情的韧性。
2. **保证金准备不足**：极端行情下交易所大幅提高保证金比例（LME 从 10% 提到 50%+），远超常规测算。
3. **流动性风险低估**：极端行情下对手方消失，无法平仓止损。
4. **跨市场套保的风险**：青山在 LME 卖空，但现货交割能力不足（标准品 vs 非标品）。
5. **场外头寸不透明**：部分头寸通过 OTC 市场，监管和风控难以穿透。

### 4.3 改进措施

- **仓位分散**：多市场、多品种、多到期日分布
- **保证金冗余**：按极端情景（非正常情景）预留
- **交割能力匹配**：确保可交割货源或提前锁仓
- **场外头寸穿透**：所有衍生品头寸统一纳入风控系统
- **应急预案**：与银行/经纪商提前约定极端行情的处理机制

## 五、套保会计与合规

### 5.1 套保关系文档

每笔套保需记录：
- 被套保项目（现货资产/预期交易）
- 套保工具（期货/期权/远期）
- 套保比率
- 套保有效性评估方法
- 无效套保部分的处理

### 5.2 套保有效性评估

**回归分析法：**
```
Δ现货价格 = α + β × Δ期货价格 + ε

套保有效条件：
- R² > 80%
- 0.8 < β < 1.25
```

**美元偏移法（Dollar Offset）：**
```
套保有效性 = Δ套保工具公允价值 / Δ被套保项目公允价值
有效区间：80% - 125%
```

## 六、经验总结

1. **套保的本质是风险管理，不是利润中心**——风控独立于业务，决策权和监督权分离。
2. **极端情景必须当真**——不是会不会发生，而是发生之后能不能活下来。
3. **保证金是命门**——极端行情下追保需求可能呈几何级增长。
4. **交割能力就是话语权**——LME 事件中，有货和没货是生死区别。
5. **穿透式风控**——所有衍生品头寸（含OTC）必须统一计量、统一监控，不能有盲区。
